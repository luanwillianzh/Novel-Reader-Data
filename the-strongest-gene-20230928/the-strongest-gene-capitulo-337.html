Início da Experiência

“Mesmo se ele for talentoso, isso não vai servir, não é? Isso é muito excessivo.”

Todos começaram a repreender o novo recruta.

Evidentemente, esse novo recruta que apareceu do nada não foi particularmente bem-vindo. Mesmo aqui, todos sabiam da importância deste novo projeto. Os níveis superiores haviam, pela primeira vez, aprovado um orçamento ilimitado para este projeto. Como tal, os membros de quase todos os outros projetos estavam competindo para ingressar neste novo projeto.

Um único olhar sobre o nível do sistema de defesa colocado em Wang Yao foi suficiente para mostrar a importância deste projeto.

Essa Wang Yao era uma pessoa que os escalões superiores haviam capturado pessoalmente para serem colocados em pesquisas. Contanto que sua pesquisa mostrasse resultados, seria uma contribuição gigantesca para a organização.

Com isso, quem não gostaria de entrar no projeto?

“Ouvi dizer que o chefe só saiu para enviar a inscrição para este projeto. Por que ele trouxe alguém de volta?”

“Qual poderia ser o motivo? Por coincidência, a organização estava realizando um processo de recrutamento. Como ele viu alguém com quem estava satisfeito ali, ele trouxe aquele cara de volta.”

“Verdade.”

Todos discutiram sem parar.

No dia seguinte, Quadrado, com o novo recruta o seguindo, voltou formalmente ao departamento.

Com isso, o projeto “MIMK-039” foi oficialmente ativado também. O principal objetivo do projeto era descobrir uma maneira de desbloquear mais potencial dentro de um ser humano. Eles pensaram que a humanidade já havia desenterrado todo o potencial disponível no corpo de um humano. No entanto, a aparência de Wang Yao aparentemente abriu as portas para um novo mundo.

Se eles pudessem descobrir os segredos de Wang Yao… Se eles pudessem descobrir o motivo pelo qual esta jovem era tão poderosa… Se…

Este foi o verdadeiro motivo pelo qual a Organização Misteriosa estava disposta a alocar uma quantidade tão grande de recursos para este projeto.

Só com isso Wang Yao mordeu a isca. Para capturá-la, a Organização Misteriosa pagou um preço extremamente alto. Como tal, uma quantia correspondente de retorno teve que ser feita agora que ela havia sido capturada.

Três dias seriam usados para preparar este projeto. Após três dias, o projeto seria oficialmente iniciado.

Este serviço de primeira linha permitiu que Wang Feng aumentasse consideravelmente sua força de maneira confortável.

“Classe C de alto nível, nada mal.” Quadrado ficou extremamente satisfeito. “Você ainda está extremamente fraco. Mas, essa força já é suficiente para você participar desse projeto.”
