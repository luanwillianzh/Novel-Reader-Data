Mensagem do Autor/Tradutor

Para responder às perguntas a respeito da fusão entre as dez classes de empregos.

A condição para a fusão das classes não são as classes básicas de trabalho. Tem que ser pelo menos as segundas classes avançadas. Por exemplo, William Feiticeiro de Gelo pode ser contado como uma dessas “Classes Avançadas”. O Feiticeiro de Gelo é a próxima evolução da classe de trabalho Mago de Gelo.

Obviamente, para atender aos requisitos do Pau Para Toda Obra, a classe Feiticeiro de Gelo deve atingir seu nível máximo primeiro.

Todas as classes de trabalho têm 3 formulários. Básico – Avançado – Final. William será capaz de cumprir a condição de Pau Para Toda Obra depois de obter 10 classes de trabalho avançadas e todos eles devem estar em seu nível máximo.

Naturalmente, se William for capaz de obter 10 classes finais e mesclá-las, o resultado será uma forma mais poderosa do Pau Para Toda Obra.

O mesmo pode ser dito para as classes de prestígio. Por exemplo, a classe de prestígio “Discípulo do Trovão” e “Príncipe das Trevas” (ooops, desculpe spoiler). Esses dois são apenas o formulário básico de trabalho das classes de prestígio.

Suas respectivas “Classes de Transcendência” são equivalentes às Classes de trabalho avançadas das profissões normais.

Quanto às classes especiais como mago de batalha, estas são consideradas uma classe de trabalho avançada. A razão é porque você precisa ter uma Classe de Trabalho de Mago (qualquer tipo de mago) e uma Classe de Trabalho de Monge em seu nível máximo para desbloquear esta Classe Especial.

Se você está familiarizado com Final Fantasy Tactics, algumas classes especiais podem ser desbloqueadas quando você atende aos respectivos requisitos.
