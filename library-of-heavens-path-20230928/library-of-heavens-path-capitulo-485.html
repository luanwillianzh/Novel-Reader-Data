Uma formação que teve sua operação interrompida é apenas uma formação que foi ‘desligada’. Ainda é possível reativá-lo depois disso.

Para resumir, é apenas a diferença de o núcleo da formação ter sido destruído ou não.

Quebrar uma formação significa destruir toda a formação e interromper uma formação significa usar um método exclusivo para interromper sua operação. Embora os efeitos fossem semelhantes em aparência, há uma enorme diferença entre os dois.
